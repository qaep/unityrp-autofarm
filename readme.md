# Système autonome de farm pour UnityRP v2

Ce projet fonctionne par vision par ordinateur via des masques hsv en utilisant opencv pour capter des faisceaux. Il est uniquement à but d'éducation et d'analyse. Il est aussi uniquement destiné au serveur UnityRP V2 de FiveM. Celui-ci gère automatiquement déplacement, détection, récolte, consommation et pose des objets récoltés dans un coffre : [Camion](#camion)

## Prérequis
[Git](https://git-scm.com/), Le reste est automatiquement installé.

##  Utilisation
Dans une invite de commande : 
```cmd
git clone https://github.com/qaep/unityrp-autofarm
```
```cmd
cd unityrp-autofarm
```
```cmd
.\start.bat
```
## Camion
Afin d'utiliser la fonctionnalité pour poser les items récoltés dans un camion ou un véhicule, il faudra impérativement que votre camion ait cette couleur rose:
### Contrôles
- **F8** : Afficher les statistiques
- **F9** : Mettre le système en pause
- **F10** : Arrêter
